# kamishibai

TRPGリプレイ動画（紙芝居動画）クリエイター。
紙芝居クリエーターで手作業していた立ち絵・メッセージウィンドウ・ステータス表示の組み立てを、
テキスト脚本から自動生成するWebアプリです。まずは迷宮キングダム向け。

## 起動

```sh
npm install
npm run dev
```

## 使い方

1. **素材ライブラリ**に立ち絵・背景・BGM・SE・ダイス連番画像を登録
   （**フォルダごとドラッグ&ドロップ／フォルダ選択**に対応。フォルダ構造はそのまま保持されます）
2. **キャラクター**を追加し、表情差分の画像・パラメータ初期値（HP/気力/配下/状態）・
   立ち絵の**表示倍率／縦位置**（素材の解像度差の補正）・**顔アイコンとその倍率**・
   **マップチップ画像とその倍率**・**ダイスセット**（連番画像のフォルダ）を設定
3. **脚本**を書くと右側にプレビューが組み上がる
4. ヘッダーの **🎬 動画書き出し** で脚本全体をmp4（1280x720/30fps、BGM/SE込み）として
   ダウンロード（WebCodecs対応ブラウザ＝Chrome/Edge推奨）

ダイスは `@dice` で連番アニメが転がってから出目を表示します。ダイス素材には
**フォルダ**（中の画像をファイル名順に連番アニメとして再生）か**画像1枚**
（転がしなしの静止ダイス）を指定できます。ヘッダーの「ダイスアニメ」チェックを
外すと、常にアニメなしで即座に出目だけ表示するモードになります。
キャラにダイスが未設定の場合は、ヘッダーで選んだ既定ダイスが使われます。

素材の選択はすべて「フォルダ → ファイル」の2段選択です。さらにキャラクターに
**素材フォルダを紐づける**と、そのキャラの立ち絵・アイコン等の選択候補が
紐づけたフォルダ内だけに絞られます（未紐づけなら全素材から選択）。

**素材はリポジトリに含めません（コミット禁止）。** 登録した素材はブラウザ内
（IndexedDB）に保存され、次回起動時に自動で復元されます。素材ライブラリの
「すべて削除」で保存データごとリセットできます。
※ブラウザやマシンをまたぐ移行は、素材フォルダを再登録してください。

脚本とキャラ設定はブラウザに自動保存されます。バックアップや別マシンへの移行は
ヘッダーの「保存（書き出し）」「開く（読み込み）」でプロジェクトファイル（.json）として
やり取りできます（素材ファイルは含まれないため、別途登録し直してください）。

### 脚本の記法

```
@bg 洞窟.png              # 背景変更
@bgm 戦闘.mp3             # BGM開始（ループ再生）
@bgm 戦闘.mp3 0.5 fade    # 音量0.5・フェードイン付き（fade=秒 で長さ指定）
@bgm stop fade=2          # 2秒かけてフェードアウトして停止
@se ダイス.wav 0.8        # 効果音（音量指定は省略可）
@show 名前 表情 left      # 立ち絵表示（セリフを書けば空いた位置に自動表示されます）
@show 名前 right flip     # 右側に左右反転で表示（flip/noflip。向かい合わせの演出）
@hide 名前                # 立ち絵消去
@damage 名前 5            # ダメージ → ステータスバーのHPに自動反映＋ポップ演出
@heal 名前 3              # 回復
@set 名前 状態 肥満       # キャラの任意パラメータ変更
@setglobal ターン 3       # 全体パラメータ（民の声・ターン等）の変更
@dice 2d6 8               # ダイス演出（連番アニメ→出目表示）
@dice 名前 2d6 8          # キャラのダイスセット（色）で振る
@map 戦場.png             # 画像の戦闘マップ/ダンジョンマップ表示（@map off で消去）
@bf                       # 標準戦場（味方本陣〜敵本陣の6列）。味方は青系・敵は暖色系
@bf 地雷原 地雷原 敵前衛 敵後衛   # 特殊戦場は列ラベルを並べる（@bf off で消去）
@trap 3 地雷原            # 戦場トラップ：列3を「地雷原」に改名＋朱色表示を1コマンドで
@trap 3 off               # 罠解除：元の名前（味方前衛など）と色に自動で戻る
@lane 2 danger            # 列の状態だけ変える（normal で解除）
@lane 2 罠原              # 列のラベルだけ変える（状態と同時指定も可: @lane 2 罠原 danger）
@chip 名前 30 60          # 画像マップ: %座標(0-100) / 生成マップ: 列 行（小数で微調整可）
@chip 名前 off            # チップ撤去
@mark 3 2 死              # 白札マーカー（死・天・鴉など。@mark 3 2 off で撤去）
@name モブ 村長           # ステータスバー等の表示名を変更（@name モブ 敵兵士 で再変更）
@status off               # ステータスバー表示切替
@wait 1.5                 # 表示時間（秒）
@fadeout 1                # シーン切替: 1秒かけて暗転（秒省略で1秒）
@fadein 1.5               # 次のカットを暗転から明転で始める（間に@bg等を挟んでシーン転換）
@fadeout 1 white          # ホワイトアウト（white/白/black/黒/#rrggbb で色指定）
@fadein 1 白              # ホワイトインで次のシーンへ（回想・爆発の定番演出）

名前: セリフ              # 1カット。立ち絵は自動表示（複数人は左→右→中央に自動配置）
名前(笑顔): セリフ        # 表情差分を自動差し替え。発言していないキャラは自動で暗くなります
```

### 別名（PL/PC・モブの使い回し）

キャラクターには**別名**を登録できます。「国王」の別名に「PL1」を入れておくと、
脚本で `PL1:` と書いても同じ立ち絵・パラメータが使われます（`@damage PL1 2` も
国王のHPに反映）。話者プレートには脚本に書いた名前がそのまま表示されるので、
ゲーム中はPC名・雑談パートはPL名と**書き分けるだけ**で切り替わります。

モブの使い回しは `@name` で。「モブ」1体を登録しておき、`@name モブ 村長` →
`@name モブ 敵兵士` とすれば、ステータスバー等の表示名がそのカット以降で変わります。

### フォント

エディタと動画のフォントは、既定でユニバーサルデザインの **BIZ UDPゴシック**
（モリサワ製、SIL OFLライセンスで同梱）を使います。動画のフォントは
ヘッダーの選択で変更できます（プロジェクトファイルに保存されます）。

設計の詳細は [docs/DESIGN.md](docs/DESIGN.md) を参照。
