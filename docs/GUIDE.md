# kamishibai 使い方ガイド

TRPGのリプレイ動画を、**脚本を書くだけ**で組み立てるツールです。
立ち絵・メッセージウィンドウ・ステータスバー・戦場マップ・BGM/SE・ダイス演出を、
1本ずつ手作業で並べる代わりに、テキストの台本から自動生成します。

このガイドは上から順に読めば、初めてでも1本作れるようになっています。
必要なところだけ読みたい人は目次から飛んでください。

## 1. 全体像 — 5つのステップ

動画ができるまでは、いつもこの流れです。

```
① 素材を登録   →  ② キャラを作る  →  ③ 脚本を書く  →  ④ プレビューで確認  →  ⑤ mp4に書き出す
  立ち絵/背景        名前・パラメータ     台本テキスト        右側で組み上がる       ヘッダーのボタン
  BGM/SE/ダイス      表情・立ち絵を紐付け  （このガイドの主役）
```

- **素材**（画像・音声）はあなたのPCから登録します。ブラウザ内に保存され、次回も自動で復元されます。
  **リポジトリ（GitHub）には一切アップロードされません。**
- **キャラ**は一度作れば、脚本で名前を書くだけで立ち絵もステータスも自動で出ます。
- **脚本**が本体です。`名前: セリフ` を並べ、演出は `@` で始まる命令で指示します。
- 書いた端から**右のプレビュー**に反映されます。再生ボタンで通しの確認もできます。
- 最後にヘッダーの **🎬 動画書き出し** でmp4（1280×720 / 30fps / 音声込み）になります。

## 2. はじめての1本（チュートリアル）

実際に手を動かして、短い掛け合いを1本作ってみましょう。所要5分です。

### ステップ1: 素材を入れる

画面右下の **素材ライブラリ** に、立ち絵や背景の画像ファイルを
**ドラッグ&ドロップ**します。**フォルダごと**放り込んでもOKで、フォルダ構造は保たれます。
BGMやSEの音声ファイル（mp3 / wav / ogg など）も同じ場所に入れます。

> 素材がまだ手元になくても大丈夫。キャラと脚本だけでも、
> 名前入りの仮チップや文字だけで動作を確認できます。

### ステップ2: キャラを作る

中央下の **キャラクター** 欄で、名前（例: `国王`）を入れて「追加」。
できたカードを開いて、

1. **立ち絵（表情差分）** の「default」に、さっき入れた立ち絵画像を選ぶ
2. 笑顔などの差分が欲しければ「表情名を追加」に `笑顔` と入れて Enter → その画像を選ぶ
3. パラメータ初期値（ＨＰ・気力・配下・状態）を必要なら直す

同じ要領で、相手役（例: `従者`）も作っておきます。

### ステップ3: 脚本を書く

左の **脚本** 欄に、こう書きます。

```
@bg 洞窟.png

国王: ここが例の迷宮だな。
従者(笑顔): わくわくしますねっ！
国王(真剣): 気を抜くな。何か来るぞ。
```

- `名前: セリフ` が1カット（1コマ）です。立ち絵は**自動で配置**されます。
- `名前(表情):` と書くと、その表情の立ち絵に**差し替わります**。
- `@bg` は背景の切り替え。ファイル名は素材ライブラリに登録した名前です。

### ステップ4: プレビューで確認

書いた瞬間から右のプレビューに反映されます。
`◀ ▶` でカットを送り、`▶ 再生` で通しの動きを確認できます。

### ステップ5: 書き出す

ヘッダーの **🎬 動画書き出し** を押すと、脚本全体がmp4になってダウンロードされます
（WebCodecs対応の Chrome / Edge 推奨）。

これで1本完成です。あとは脚本にBGMやダメージ演出、戦場マップを足していくだけ。
次章からは「やりたいこと」ごとの書き方を説明します。

## 3. 脚本の基本ルール

覚えることは3つだけです。

### ① セリフは `名前: テキスト`

```
国王: いくぞ。
```

- **全角コロン `：`** でも書けます。
- 立ち絵は登場順に**左 → 右 → 中央**へ自動配置。発言していないキャラは自動で少し暗くなります。
- `名前(表情):` で立ち絵を差し替え。表情名はキャラ設定で登録したものです。

### ② 演出命令は `@` で始まる行

```
@bg 森.png
@bgm 戦闘.mp3
@wait 2
```

1行に1命令。全角の `＠` でも認識します。命令の一覧は[6章の早見表](#6-コマンド早見表)にまとめてあります。

### ③ コメントは `#`

```
# これはメモ。カットにはなりません
@bg 城.png   # 命令行の途中にも書けます
```

- 行頭の `#`（全角 `＃` も可）はまるごとコメント。
- **命令行（`@`）**では、途中の ` #` から後ろもコメントになります。
- ただし**セリフ行の途中の `#` はコメントになりません**（「HP#1」のような台詞を壊さないため）。

## 4. やりたいことから引く（逆引きレシピ）

### 🎵 BGMを流したい・止めたい

```
@bgm 戦闘.mp3            # 開始（自動でループ再生。次の@bgmか@bgm stopまで鳴り続ける）
@bgm 戦闘.mp3 0.5        # 音量を半分（0〜1で指定）
@bgm 戦闘.mp3 0.8 fade   # フェードインしながら開始
@bgm 日常.mp3           # 別の曲に切り替え（前の曲は止まる）
@bgm stop               # 停止
@bgm stop fade=2        # 2秒かけてフェードアウトして停止
```

`fade` は既定1.5秒。`fade=秒` で長さを指定できます。

### 🔊 効果音（SE）を鳴らしたい

```
@se 剣撃.wav            # そのカットの頭で1回鳴る
@se 剣撃.wav 0.7        # 音量指定（省略可）
```

`@se` の直後に `@dice` を置くと、**ダイスの転がりがSEの長さに自動で合います**（後述）。

### 💥 ダメージ・回復・パラメータ増減

数値は**ステータスバーに自動反映**され、ポップ演出も出ます。

```
@damage 国王 5          # HPを5減らす
@damage ゴブリンA ゴブリンB 5  # 複数人にまとめて同じ値（敵味方の混在OK）
@damage 味方 3 敵 8           # 「名前… 数値」を並べると個別の値も指定できる
@heal 国王 4            # 回復（書き方は@damageと同じ、複数人可）
@mod 気力 国王 -1 従者 +2  # HP以外（気力など）の増減。符号付きで複数人
@set 国王 状態 麻痺     # 任意パラメータを好きな値に（状態異常など）
```

- `@damage` / `@heal` の数値は**正の数**。
- `@mod` は `+2` / `-1` のように**符号付き**（0以外）。上限・下限は自動でクランプされます。

### 🎲 ダイスロールを見せたい

```
@dice 2d6 8              # 連番画像が転がる → 出目「8」を表示
@dice 国王 2d6 8   # そのキャラ専用のダイス（色）で振る
@se ダイス.wav
@dice 2d6 8             # ↑直前に@seを置くと、転がりの長さがSEに揃う
```

- ダイス素材には、**フォルダ**（中の画像をファイル名順に連番アニメ再生）か
  **画像1枚**（転がしなしの静止ダイス）を指定できます。キャラ設定の「ダイス」欄で選びます。
- ヘッダーの「**ダイスアニメ**」チェックを外すと、常にアニメなしで即座に出目だけ出ます。
- キャラにダイス未設定なら、ヘッダーの「既定ダイス」が使われます。

### 🗺️ 戦場マップを出したい（画像なしでOK）

素材が要らない**生成戦場**が便利です。

```
@bf                     # 標準の6列（味方本陣/味方後衛/味方前衛/敵前衛/敵後衛/敵本陣）
                        # 味方＝青系・敵＝暖色系で自動に色分け
@bf 地雷原 味方前衛 敵前衛 敵本陣  # ラベルを並べれば好きな戦場に
@bf off                 # 消す
```

画像の戦闘マップ・ダンジョンマップを使いたいときは `@map`。

```
@map ダンジョン.png      # 画像マップを表示
@map off                # 消す
```

### 🧩 マップにキャラ/敵のコマ（チップ）を置きたい・動かしたい

```
@chip 国王 3 2         # 生成戦場は「列 行」（小数で微調整可）
@chip 国王 30 60       # 画像マップは「% 座標」（0〜100）
@chip ゴブリンA 4 2 ゴブ.png  # 登録していない敵もOK。4番目に画像を指定
@chip 国王 4 2         # 同じ名前を置き直す＝移動。次のカットで滑らかにスライド
@chip 国王 off         # 撤去
```

**チップの見た目の決まり方**（上から順に、あるものが使われる）:

1. `@chip` の4番目で指定した画像
2. キャラ設定の「マップチップ」画像
3. キャラ設定の「顔アイコン」
4. どれもなければ**既定のコマ** — **味方＝青い丸 / 敵＝朱色の角丸四角**（名前の先頭2文字入り）

敵味方は「**ステータスバーに載っている登録キャラ＝味方**、それ以外＝敵」で自動判別します。
これは下の「敵の3段階」の考え方とそのまま対応します。

### 🐉 敵のステータスをどこまで管理するか（3段階）

戦闘ごとに敵の数や種類が変わるので、手間に応じて使い分けます。

| 敵のランク | やること | 記録 | 見た目 |
|---|---|---|---|
| 雑魚 | `@chip 名前 列 行 画像` で置くだけ | しない | 朱色のコマ |
| 中ボス | キャラ登録して「ステータスバー表示」を**OFF** | HPだけ | コマの下にミニHPバー（`@damage`が反映） |
| ボス | 普通に登録して「ステータスバー表示」ON | フル | ステータスバーに表示 |

### ⚔️ トラップで戦場の一部を変えたい

```
@trap 3 地雷原          # 列3を「地雷原」に改名＋朱色の危険表示を1発で
@trap 3 off             # 罠解除：元の名前と色に自動で戻る
@lane 2 danger          # 列の状態だけ危険色に（normalで解除）
@lane 2 罠原            # 列のラベルだけ変える（@lane 2 罠原 danger で同時指定も）
@mark 3 2 死            # マス(列3,行2)に白い札マーカー（@mark 3 2 off で撤去）
```

### 🏰 ダンジョン地図・王国の土地を出したい（素材不要）

```
@dungeon 深き渓谷 6x6      # 開始。全マスが「未探索の白箱」になる（3x3/6x6など自由）
@room 2 1 北の館 敵2 罠0     # 探索で開示：列 行 [名前] [カウンタ…]
@room 3 3 深き渓谷 2x2 敵7 罠1   # 2x2 のセル結合で大部屋
@room 5 5                     # 探索したが「何もない部屋」だった
@room 2 1 敵-1                # 符号付き＝増減（敵を1体倒した）。「敵0」のような無符号は設定
@link 2 1 3 1                 # 部屋間の通路。斜めも可。線は部屋の壁で止まる（部屋名と被らない）
@link 2 1 上                  # 外部からの入口（上下左右）。外縁の部屋にダンジョン外からの矢印
@dungeon off                  # 消す
```

- カウンタのラベルは自由（敵・罠・宝など）。増減しても0未満にはなりません
- **王国の土地も同じ書式**です：`@dungeon 王国 3x3` ＋ `@room 1 1 王宮`（名前だけ書く）
- 現在地は `@chip 名前 列 行`（顔アイコンが乗る）、白札は `@mark`

### 🌍 王国周辺図（色分き領土マップ）を出したい

```
@kingdom 北方領地 6x6
@terr 1 2 自国領/古い神殿 自国    # 列 行 名前 [自国|味方|敵]。名前は / で改行
@terr 6 4 遺跡             # 所属なし＝黒文字（未購入・攻略対象の土地）
@dist 5 4 3                     # 出発地→攻略対象のマス数（＝イベント表を振る回数）
@terr 6 4 自国領/遺跡 自国  # 再指定で上書き（攻略・買収した等）。@terr 6 4 off で撤去
@kingdom off
```

- 所属の色は色覚多様性に配慮した配色（自国=青緑・味方=青・敵=朱）で、敵国は**破線枠**でも区別
- `@dist` の数字は空きマスなら中央に大きく、領土のあるマスなら右下に小さく重なります

### 🎬 シーンを転換したい（フェード）

```
@fadeout 1              # 1秒かけて暗転（秒を省くと1秒）
@fadein 1.5             # 次のカットを暗転から明転で始める
@fadeout 1 white        # ホワイトアウト（white/白/black/黒/#rrggbb で色指定）
@fadein 1 白            # ホワイトインで次のシーンへ（回想・爆発の定番）
```

暗転と明転の間に `@bg` などを挟めば、きれいな場面転換になります。

### 🖼️ ロゴや一枚絵を全画面で見せたい

```
@still ロゴ.png 3        # 一枚絵を3秒だけ全画面表示（立ち絵などは隠れる）
@still ロゴ.png 音.wav   # 音声付き。表示時間＝音声の長さに自動一致（秒指定で上書き可）
@still ロゴ.png 3 black  # 背景色を指定（省略時は白）
@still off               # 解除して通常シーンに戻る
```

### 📜 黒背景・白背景に文字を並べたい（お宝表・キャラ紹介）

```
@text black             # テキスト画面の開始（white / #rrggbb も可）
@c ～お宝表～            # 行頭に @c を付けた行は中央寄せ
                        # （空行）はそのまま行間になる
天狗：
　木の素材10個           # 全角スペースの字下げも保持される
@end                    # ブロックの終わり
@text off               # テキスト画面を解除
```

文字色は背景の明るさから白/黒が自動で選ばれます。`@text` の間も立ち絵は表示され続けます。

### 🔠 サムネ用に文字を大きくしたい

```
@fontsize 2.5           # メッセージ文字を2.5倍に（0.5〜5）
@fontsize off           # 元に戻す（@fontsize 1 でも可）
```

1.4倍を超えると、視認性のため自動で太字＋縁取りになります。

### 🎭 モブの表示名を使い回したい / PL名とPC名を切り替えたい

**モブの使い回し** — 「モブ」1体を登録し、`@name` で表示名を変えます。

```
@name モブ 村長          # このカット以降、表示名が「村長」に
@name モブ 敵兵士        # 次の場面ではこの名前に
```

**PL名 ↔ PC名** — キャラの「別名」に登録しておくと、脚本でどちらの名前を書いても
同じキャラとして扱われ、**話者プレートには書いた名前がそのまま出ます**。

```
# 国王の「別名」に「PL1」を登録しておくと…
PL1: 今日はよろしくね     # 雑談パートはPL名で
国王: いくぞ！            # 本編はPC名で（同じ立ち絵・同じHP）
@damage PL1 2            # 別名でもちゃんと国王のHPに反映される
```

### 立ち絵をこまかく制御したい

セリフを書けば自動で出ますが、明示的に置くこともできます。

```
@show 国王 笑顔 left   # 左に「笑顔」で表示
@show 従者 right flip        # 右に左右反転で表示（向かい合わせの演出。flip / noflip）
@hide 国王             # 消す
```

### 間を取りたい・表示時間を変えたい

```
@wait 2                 # このカットを2秒見せる（既定は約2.5秒）
```

## 5. キャラクター設定のすべて

キャラクターカードを開くと、次の項目を設定できます。

| 項目 | 役割 |
|---|---|
| **ステータスバー表示** | ONで下部のステータスバーに載る（＝味方扱い）。敵はOFFにするとコマ下にミニHPバー |
| **別名** | PL名・変名など。読点/カンマ区切りで複数。脚本でどれを書いても同一キャラ扱い |
| **素材フォルダの紐づけ** | 紐づけると、このキャラの素材選択候補がそのフォルダ内だけに絞られる（大量素材でも選びやすい） |
| **立ち絵（表情差分）** | `default` と、追加した表情ごとに画像を割り当て。`名前(表情):` で切り替わる |
| **顔アイコン** ＋倍率 | ステータスバーやチップに使う小さな顔。倍率で素材の解像度差を補正 |
| **マップチップ** ＋倍率 | 戦場コマ専用の画像（未指定なら顔アイコンを流用） |
| **右側配置で左右反転** | 右に置いたとき絵を反転（向かい合わせ用） |
| **立ち絵倍率 / 縦位置(px)** | 立ち絵の大きさと上下位置の微調整（素材ごとの解像度差を吸収） |
| **ダイス** | このキャラ専用のダイスセット（フォルダ＝連番アニメ / 画像1枚＝静止） |
| **パラメータ初期値** | ＨＰ・気力・配下・状態などの開始値（テンプレートで定義） |

### 迷宮キングダムの初期パラメータ

- キャラ: **ＨＰ 10 / 気力 3 / 配下 1 / 状態 正常**
- 全体（王国）: **民の声 10 / ターン 1 / クォーター 1 / 食事 0 / 生産力 0**

全体パラメータは `@setglobal ターン 3` のように変更できます。

## 6. コマンド早見表

| コマンド | 意味 |
|---|---|
| `名前: セリフ` | 1カット。立ち絵は自動配置。`名前(表情):` で表情差し替え |
| `@bg 画像` | 背景を変える |
| `@bgm 曲 [音量] [fade\|fade=秒]` | BGM開始（ループ）。`@bgm stop [fade=秒]` で停止 |
| `@se 音 [音量]` | 効果音を1回鳴らす |
| `@show 名前 [表情] [left\|center\|right] [flip\|noflip]` | 立ち絵を明示表示 |
| `@hide 名前` | 立ち絵を消す |
| `@damage 名前… 数値 [名前… 数値…]` | HPを減らす（複数人・個別値可） |
| `@heal 名前… 数値 […]` | 回復（書き方は@damageと同じ） |
| `@mod パラメータ 名前… ±数値 […]` | HP以外の増減（気力など。符号付き） |
| `@set 名前 パラメータ 値` | 任意パラメータを指定値に |
| `@setglobal パラメータ 値` | 全体パラメータ（民の声・ターン等）を変更 |
| `@dice [名前] NdM 出目` | ダイス演出（連番アニメ→出目） |
| `@map 画像` / `@map off` | 画像の戦闘/ダンジョンマップ |

## 7. 保存・バックアップ・書き出し

- **自動保存**: 脚本とキャラ設定はブラウザに自動保存されます。次回開くと続きから作業できます。
- **素材の保存**: 登録した素材はブラウザ内（IndexedDB）に保存され、次回自動復元。
  素材ライブラリの「**すべて削除**」で保存データごとリセットできます。
- **プロジェクトの持ち出し**: ヘッダーの「**保存（書き出し）**」で `.json` に、
  「**開く（読み込み）**」で復元。**素材ファイルは含まれない**ので、別マシンでは素材を登録し直してください。
- **動画書き出し**: ヘッダーの「**🎬 動画書き出し**」でmp4（1280×720 / 30fps / BGM・SE込み）。
  Chrome / Edge 推奨です。
- **フォント**: 既定はユニバーサルデザインの **BIZ UDPゴシック**（同梱）。
  ヘッダーの選択で変更でき、プロジェクトファイルに保存されます。

> **重要**: 素材（画像・音声）は GitHub には保存されません。
> あくまであなたのブラウザとPCの中だけにあります。

## 8. よくある質問とつまづき

**Q. 立ち絵が出ない**
キャラ名の綴りが脚本と一致しているか、そのキャラの「立ち絵（default）」に画像を割り当てたかを確認してください。別名で書いている場合は、その別名がキャラに登録されているか確認を。

**Q. 背景やBGMが反映されない**
`@bg` / `@bgm` のファイル名が、素材ライブラリに登録した名前と一致しているか確認してください。フォルダごと入れた場合、同名ファイルはフォルダ名まで含めて指定が必要なことがあります。

**Q. 「解釈できない行です」と出る**
セリフは必ず `名前: テキスト` の形にしてください（コロンが必要）。命令のつもりなら行頭に `@` を付けます。

**Q. ダメージが反映されない / 警告が出る**
`@damage 名前 数値` の名前が登録キャラ（または別名）と一致しているか確認を。未登録名は警告になります（チップだけなら未登録でも置けます）。

**Q. 敵味方の色を変えたい**
既定コマは「ステータスバー表示ON＝味方（青丸）/ それ以外＝敵（朱四角）」で自動判別します。専用のチップ画像や顔アイコンを設定すれば、その絵が優先されます。

**Q. 書き出しボタンが効かない / 失敗する**
mp4書き出しは WebCodecs 対応ブラウザ（Chrome / Edge）が必要です。Safari など未対応環境では動作しないことがあります。

**Q. 別のPCに移したい**
「保存（書き出し）」で `.json` を持ち出し、移行先で「開く（読み込み）」。素材は含まれないので、移行先で素材フォルダを登録し直してください。
